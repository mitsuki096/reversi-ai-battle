# Reversi AI Battle

Reversi AI Battle は、ブラウザ上でユーザーとCPUがリバーシで対戦できるWebアプリケーションです。  
ユーザーは黒石、CPUは白石として対戦します。

ユーザーが盤面のマスをクリックすると、アプリが合法手判定、石の反転処理、CPUのAI思考、パス判定、勝敗判定を行い、結果を画面に表示します。

## AI難易度

| 難易度 | 内容 |
| --- | --- |
| Easy | 深さ2のαβ探索 |
| Normal | 深さ4のαβ探索 |
| Hard | 深さ6のαβ探索 |
| Ultimate | 反復深化探索、αβカット、手の並び替え、評価関数強化、終盤完全読み、置換表、時間制限付き探索 |

Ultimateでは、できる限り強い手を選択するために、複数の探索技術を組み合わせています。  
CPU思考時間は最大3秒程度を目安としています。

## おすすめマス機能

ユーザーが置ける合法手の中から、おすすめのマスを黄色い `★` で表示します。

おすすめマスは、以下の要素をもとに評価しています。

- 角を取れるか
- 辺に置けるか
- 危険マスを避けられるか
- CPUの合法手を減らせるか
- 自分の合法手を確保できるか
- 石数差
- 終盤の局面評価
- 浅いαβ探索による先読み

おすすめマス表示は、画面上のボタンからON/OFFを切り替えることができます。  
初期状態では、おすすめマス表示はOFFです。  
また、CPU思考中、パス時、ゲーム終了時にはおすすめマスを表示しないようにしています。

## Ajax対応

本アプリでは、盤面クリック時に画面全体を再読み込みせず、Ajaxで盤面情報のみを更新します。

これにより、以下のような操作性を実現しています。

- 一手ごとに画面が上に戻らない
- 盤面のちらつきが少ない
- 自分の石を先に表示してからCPU思考に入る
- CPU思考中に `CPU thinking...` を表示
- CPU着手後に盤面だけを更新

## 実行方法

### Dockerで起動する場合

事前に Docker Desktop を起動しておきます。

プロジェクトフォルダで以下を実行します。

```bash
docker compose up --build
```

ビルドと起動が完了すると、Spring Bootアプリケーションが8080番ポートで起動します。

ブラウザで以下のURLにアクセスしてください。

```text
http://localhost:8080
```

### 停止方法

起動中のターミナルで以下を押します。

```text
Ctrl + C
```

コンテナを停止・削除する場合は、以下を実行します。

```bash
docker compose down
```
