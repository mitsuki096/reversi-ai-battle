# Reversi AI Battle

Reversi AI Battle は、ブラウザ上でユーザーとCPUがリバーシで対戦できるWebアプリケーションです。  
ユーザーは黒石、CPUは白石として対戦します。

ユーザーが盤面のマスをクリックすると、合法手判定、石の反転処理、CPUのAI思考、パス判定、勝敗判定を行い、結果を画面に表示します。

---

## 主な機能

- 8×8のリバーシ盤面表示
- 初期配置の表示
- ユーザー対CPUのリバーシ対戦
- ユーザーは黒石、CPUは白石
- 合法手判定
- 石の反転処理
- CPU自動着手
- パス判定
- 自分が置けない場合のパス処理
- CPUが置けない場合の自動パス処理
- ゲーム終了判定
- 勝敗判定
- 黒石・白石数の表示
- 合法手ハイライト
- CPU最後の着手位置の強調表示
- CPU評価値の表示
- CPU思考時間の表示
- 探索深さの表示
- 難易度選択
- リセット機能
- おすすめマス表示
- おすすめマス表示のON/OFF切り替え
- Ajaxによる盤面更新
- CPU思考中表示
- モダンなダークテーマUI
- Dockerによる起動対応

## AI難易度

| 難易度 | 内容 |
| --- | --- |
| Easy | 深さ2のαβ探索 |
| Normal | 深さ4のαβ探索 |
| Hard | 深さ6のαβ探索 |
| Ultimate | 反復深化探索、αβカット、手の並び替え、評価関数強化、終盤完全読み、置換表、時間制限付き探索 |

Ultimateでは、できる限り強い手を選択するために、複数の探索技術を組み合わせています。  
CPU思考時間は最大3秒程度を目安としています。

## おすすめマス機能

ユーザーが置ける合法手の中から、おすすめのマスを黄色い `★` で表示します。

おすすめマスは、以下の要素をもとに評価しています。

- 角を取れるか
- 辺に置けるか
- 危険マスを避けられるか
- CPUの合法手を減らせるか
- 自分の合法手を確保できるか
- 石数差
- 終盤の局面評価
- 浅いαβ探索による先読み

おすすめマス表示は、画面上のボタンからON/OFFを切り替えることができます。  
また、CPU思考中やゲーム終了時にはおすすめマスを表示しないようにしています。

---

## Ajax対応

本アプリでは、盤面クリック時に画面全体を再読み込みせず、Ajaxで盤面情報のみを更新します。

これにより、以下のような操作性を実現しています。

- 一手ごとに画面が上に戻らない
- 盤面のちらつきが少ない
- 自分の石を先に表示してからCPU思考に入る
- CPU思考中に `CPU thinking...` を表示
- CPU着手後に盤面だけを更新
